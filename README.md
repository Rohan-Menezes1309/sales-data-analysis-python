# Sales Data Analysis Using Python

## Project Overview

This project analyzes sales and profitability data to identify key business trends and performance drivers.

The dataset includes transaction-level information such as sales, profit, discount, shipping cost, and product categories.

---

## Tools Used

1. Python
2. pandas
3. matplotlib
4. seaborn

---

## Key Insights

1. Central region generated the highest sales, while Canada contributed the least
2. Technology category was the most profitable, while Furniture showed lower profitability
3. Sales peaked in December and were lowest in February, indicating strong seasonality
4. Phones and Copiers were the top-performing sub-categories
5. Higher discounts were associated with lower profitability and increased losses
6. Shipping cost showed no strong relationship with profit

## Conclusion

The analysis highlights the importance of managing discount strategies, focusing on high-performing regions and categories, and understanding seasonal demand patterns to improve overall business performance.
